1. Introduction

This project presents a fully-featured Library Management System built as the OOP Complex Engineering Problem for course SW121. The application is developed using Java Swing for the graphical user interface and MySQL as the backend relational database, connected via JDBC.

The system enables library staff to manage books, members, and book issue/return transactions through a clean, role-based interface supporting both Admin and standard User roles.

2. Requirements Coverage

Req 1
Login / Authentication
LoginFrame with Admin and User roles. Credentials validated against MySQL users table via UserDAO.
Req 2
CRUD Operations
Full Create / Read / Update / Delete on Books, Members, IssuedBooks, and Users through dedicated DAO classes.
Req 3
Search & Filter
Live keyword search on every panel using SQL LIKE queries. Category filter dropdown on Books panel. Status filter on Issue panel.
Req 4
Data Validation
Validator utility class performs UI-level checks (empty fields, email/phone format, year range). Database-level constraints via FK, UNIQUE, and NOT NULL.
Req 5
Reports Generation
ReportsPanel displays issue history with filter. Print button uses JTable.print() with header/footer for direct printer output.
Req 6
Attractive GUI
UITheme utility provides a consistent blue/dark-sidebar design with rounded buttons, alternating table rows, gradient panels, and card layouts.
Req 7
Database Connectivity
DBConnection singleton manages JDBC connection to MySQL. All queries use PreparedStatement to prevent SQL injection.
Req 8
Modular Code Design
Clean MVC-like package separation: models/, dao/, ui/, utils/, database/. Every class has Javadoc comments.

5. Installation & Setup

1
Install Prerequisites
Ensure JDK 17+, MySQL 8+, and NetBeans / IntelliJ IDEA are installed on your machine.
2
Create the Database
Open MySQL Workbench or the MySQL CLI and run the provided SQL file:
source LibraryMS/database/library_schema.sql;
This creates the library_db database, all tables, and sample data.
3
Add MySQL JDBC Driver
Download mysql-connector-j-8.x.jar from dev.mysql.com/downloads/connector/j/. In NetBeans: right-click project → Properties → Libraries → Add JAR/Folder.
4
Configure DB Credentials
Open src/database/DBConnection.java and update USER and PASSWORD to match your MySQL installation.
5
Build & Run
Open the project in NetBeans → Clean and Build → Run Main.java. The Login screen appears.
